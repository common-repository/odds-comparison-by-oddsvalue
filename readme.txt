== Description ==
The OddsValue Odds Comparison plugin enables you to show structured tables with odds from all the popular football (soccer) matches on your site. It also supports tennis, ice hockey, baseball, handball and many more sports.

Odds are delivered with delay of less than 5 minutes to ensure always perfect information for your users.

The plugin is responsive and works on all  devices.

You are able to define your own fonts and colors to skin it perfect for your site. 

Pure javascript solution with no iframe integration.

== Installation ==
1. Upload the "Odds Comparison" to the folder: /wp-content/plugins/ directory
2. Activate the "Odds Comparison" through the "Plugins" menu in WordPress
3. Configure the plugin from Settings > Odds Comparison in your admin menu
4. Use [odds_comparison] shortcode where you want to display odds
